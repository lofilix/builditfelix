# builditfelix.dev

My personal portfolio site. Built with Next.js and deployed on Vercel.

🔗 **Live site:** [builditfelix.dev](https://builditfelix.dev)

![Portfolio screenshot](./public/og-image.png)

## Stack

- **Framework** — Next.js (App Router)
- **Language** — TypeScript
- **Styling** — TailwindCSS
- **Deployment** — Vercel
- **Fonts** — Silkscreen, Outfit, JetBrains Mono (Google Fonts)

## Features

- Animated hero with pixel-art identity and floating CTAs
- 3D phone and monitor mockups with parallax tilt on mouse move
- Scroll-triggered reveal animations via IntersectionObserver
- Fully responsive, mobile-first layout
- Dark theme with custom orange accent palette

## Running locally

\`\`\`bash
git clone https://github.com/felixjunedesilva/builditfelix.dev.git
cd builditfelix.dev
npm install
npm run dev
\`\`\`

Open [http://localhost:3000](http://localhost:3000).
